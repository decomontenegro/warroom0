## 🐛 Sobre o Erro do Chrome

O erro que você viu:
```
hostname_check.fd55cc53.js:28 [content-script|error]: Error sending to background hostname check
```

**Não é um problema do War Room!** É uma extensão do Chrome tentando injetar código. 

### Soluções:
1. **Modo Incógnito**: Abra em janela anônima (Ctrl+Shift+N)
2. **Desabilitar extensões**: chrome://extensions/
3. **Outro navegador**: Firefox, Safari, Edge

## 💡 Como Usar

### 1. Chat Individual
- Clique em qualquer especialista
- Converse como no WhatsApp
- Cada um tem sua expertise

### 2. Todos Juntos
- Clique em "👥 Todos os Especialistas"
- Faça uma pergunta
- Veja 5-7 especialistas respondendo

### 3. Resumo Inteligente
- Clique em "📊 Resumo Inteligente"
- Use o slider para mais/menos detalhe
- Peça resumos específicos

### 4. Prompt Builder
- Clique em "🔧 Prompt Builder"
- Selecione tópicos (Frontend, Backend, etc.)
- Use templates ou escreva seu prompt
- Veja sugestões de especialistas
- Envie para múltiplos agentes
